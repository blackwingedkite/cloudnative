# cloudnative
## 前端：旻宸、灝融、宥圻
- Build Tool: Vite

- Frontend Library: React

- CSS Library: Tailwind

- Router: React Router

- API integration: Axios

- Form validation and management: React-hook-form

- Package Manager: yarn

## 後端：立曄、柏辰、啟湘
- Express.js
- Node.js
## 資料庫
